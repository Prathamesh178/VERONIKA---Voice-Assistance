# VERONIKA---Voice-Assistance
VERONIKA is a Python-based intelligent voice assistant capable of performing tasks through voice commands such as opening apps, playing music, Google/YouTube search, telling time/date, and greeting users. It supports English, Hindi, and Marathi with clear text-to-speech responses.
